# Software

## Application - Software that lets you do something such as a web browser to surf the web or spotify to listen to music or even visual studios to write code.
## Utility - Software that lets you fix or modify your computer in some way. This is things like Antivirus or disk defrag.
## Operating System Software - O/S software is the part of a computer that manages all the resources in a computer, Provides the user interface, and the platform to build and develop applications. Examples are Windows, Apple OS X, and Linux
